## Using Docker

1. [Install Docker](https://docs.docker.com/get-docker/) on your machine.
1. Build your container: `docker pull demchukvolodymyr/dept`.
1. Run your container: `docker run -p 3000:3000 demchukvolodymyr/dept`.
